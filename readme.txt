== Description ==
Integrate Turn14 with WooCommerce

== Installation ==
1. Upload 'turn14-for-woocommerce' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your Turn14 API key and secret into Turn14->Settings tab
